Publish (a website) or Perish

May 15, 2018 1 min read

For my first post I’d like to encourage other R users to develop their own static websites using the blogdown package.

Feel free to use the code I’ve developed on my github account, but consider some better examples, including Dr Hill’s and Dr Cho Yang’s websites using the same academic framework.

Register with Github and install Git to version control and then deploy your website using Netlify. Follow Dr Hill’s recommendation for a free domain name with Github. I’ve decided to use a godaddy account for a very small fee.

Showcase your publications or methods that would otherwise not see the light of day!
